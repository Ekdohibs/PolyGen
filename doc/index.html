Polyhedral compilation

Table of contents

General-purpose libraries

  • Misc: extensions to the Coq standard library.
  • Linalg: definitions of vectors and operations over them.
  • Result: the error monad.
  • TopoSort: topological sort.
  • Mymap: workaround a bug in Coq.
  • ImpureOperations: defines sequence and mapM.
  • ImpureAlarmConfig: the error monad used throughout the code.

Polyhedral operations

  • VplInterface: interface to the VPL.
  • Canonizer: simplify redundant constraints in polyhedra.
  • Heuristics: heuristics for some polyhedral operations.
  • Projection: compute polyhedral projections.

Languages and semantics

  • Semantics: memory states, semantics for iterating over a list and associated theorems.
  • Instr: semantics of base instructions.
  • PolyLang: semantics of the scheduled polyhedral language.
  • PolyLoop: semantics of the intermediate language: AST, but each loop and guards over polyhedra.
  • Loop: semantics of the loop language.

Compilation

  • CodeGen: code generation from PolyLang to PolyLoop.
  • LoopGen: code generation from PolyLoop to Loop.
